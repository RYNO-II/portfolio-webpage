However, I can use this space to talk about
how this page was made! Everything you
see is made using SVGs instead of images.
This was done to keep all the page elements looking sharp at any screen resolution and to improve site responsiveness. Even the flowing cape is a bunch of SVG paths animated using GSAP.

The water flowing effect is made using the
SVG feTurbulence filter primitive with a
gradient applied to it to make the wave effect more noticeable .
